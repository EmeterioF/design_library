<h1>This is a library for gsap designs that you could use to create cool animations within your app or landing pages</h1>
<br>
<h2>Techonologies used:</h2>
<ul>
  <li>Typescript</li>
  <li>Tailwind</li>
  <li>GSAP</li>
</ul>
<br>
<h2>How to use</h2>
<p>Just run npm install<br/>each components are gsap animations you can use, just switch the component in the App.tsx to preview each animations</p>
